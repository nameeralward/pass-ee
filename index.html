Smart Dashboard

A personalized dashboard featuring time-based greetings with motivational quotes, daily study streak tracking, and comprehensive exam preparedness metrics to monitor your learning progress.

Quick Study Tools

Quick-access cards for essential study tools: Saved Summaries, Incorrect Answers Practice, Bookmarked Questions, and Resume Quizzes—all designed for efficient exam preparation.

Customizable Quizzes

A comprehensive quiz interface offering Mock Test and Study Mode options, with customizable question counts, time limits, and collection-based content selection for personalized learning.

Interactive Questions

Interactive exam-style questions featuring strike-through option for elimination, answer selection, submit button, detailed explanations, bookmarking, and feedback

Self-Paced Learning

A self-paced learning interface with detailed explanations, text-to-speech support, AI assistance, bookmarking, feedback options, and image support for comprehensive question review.

Comprehensive Explanations

Detailed explanations with structured sections covering correct answers, misconceptions, memory tips, and pharmacy references, presented in an expandable sheet

Advanced Text-to-Speech

Advanced text-to-speech system offering AI and system voices, customizable reading speeds, sequential reading, and smart content filtering with comprehensive accessibility settings.

AI Learning Assistant

AI-powered learning assistant offering personalized feedback, incorrect answer analysis, comprehensive topic summaries, and study planning—all powered by OpenAI's ChatGPT for pharmacy students.

AI Topic Summaries

AI-powered comprehensive topic summaries providing key concepts, detailed explanations, study points, and pharmacy references—all organized in a structured, easy-to-digest card format.

Personal Settings

Personal settings hub featuring user profile management, subscription status, exam date settings, study preferences, notification controls, and app customization with account management options.
